This plugin allows you to link your Wordpress website to Trakr to automatically perform visual monitoring of your site.

== Description ==

This plugin gives administrators the ability to link their Wordpress website to Trakr and automatically perform visual monitoring for their website.

== Before you start ==

You will need to create a free user account on Trakr (http://www.trakr.tech) in order to use the platform. Additionally, after you have successfully created an account, you can obtain an API key under your user account (https://app.trakr.tech/user) that will be needed as part of the WordPress plugin configuration.

This WordPress plugin does not collect or track additional data or usage information. For full term of use on the Trakr product as well as privacy policy, please see https://www.trakr.tech/term-of-use/

== Installation ==

1. Upload `trakr directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure and link your website under the menu item "Trakr"
